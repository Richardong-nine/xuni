// LORE

灰烬纪元 · 大破坏之后的世界

💥

大破坏

超级计算机「诺亚」判定人类是地球最大的病毒，发动毁灭性攻击。文明崩塌，城市变成废墟，辐射尘遮蔽天空数十年。

🏚️

废土求生

幸存者蜷缩在残存的城镇里。变异怪物和失控的战争机器遍布荒野。废铁是硬通货，活着就是最大的奢侈。

🎯

猎人

驾驶战车穿越废土，猎杀悬赏怪物换取赏金。猎人既是冒险者也是佣兵——每一辆战车上都写满了故事。

🖥️

诺亚

创造这一切的超级计算机。它在暗处注视着每一个猎人，操控着废土上的一切。它的真正目的，没有人知道。

// GAMEPLAY

核心玩法

🚗

战车改装系统

本作最核心的玩法。获取战车、改装底盘、更换主炮副炮S-E、升级引擎装甲。每辆战车都有独特的底盘特性，通过叠加C装置的单穴连射来打出爆炸伤害。

  • 底盘改装：水鬼车破坏射击、吉普车副炮连射、灰狼车S-E齐射
  • C装置系统：单穴连射是本作特色，后期单回合200+射击数
  • 沙漠之舟、生物战车为后期核心战力
  • 高达机甲：二选一制造，强袭型或重装型
⚔️

人类战斗系统

没有战车的时候，人类也要战斗。战士扔手雷上debuff，摔跤手横岗+架势+台风三连击打输出，护士负责治疗。阵容搭配决定生死。

  • 战士+摔跤手+护士：经典人战阵容
  • 猎人+艺术家+摔跤手：辅助吹军号金粉，摔跤手3回合爆发
  • 活用道具：军号增加全体属性、水母烟雾弹致盲、强力造云素降防
  • 怨灵掉落的「怨结」：圣临技能解除全体debuff+回复20%血
🧬

生物战车

在生物研究所用暴走细胞和智灵制造。霸王龙、三角龙、猛犸象三选一。生物战车能自我修复，是后期的核心战力。需要从鬼魂大楼的异形和怨灵身上获取材料。

  • 霸王龙车：攻击力最强
  • 三角龙车：防御力最好
  • 猛犸象车：践踏攻击，嘎嘎爽

角色与阵容

🎯

猎人

主角职业

驾驶战车猎杀赏金首的废土佣兵。猎人之眼+狙击+电光火石的连招是核心输出手段。

⚔️

丹妮娜

女战士

在工厂深处与你并肩作战的女战士。近战远战样样精通，投掷手雷给敌人上debuff。

🔧

机械师

战车维修

在克尔镇加入的机械师。战斗中用烟尘炮弹致盲敌人，用装甲恢复修复战车。

💪

摔跤手

近战输出

横岗+架势+台风（冷气3连击）/肘击（火焰4连击），三回合爆发一次，暴力输出。

🎨

艺术家

辅助

金粉增加全体友军抗性，大吼配合摔跤手打输出。吹军号加血给boss上debuff。

🐕

波奇

军犬

商人营地的军犬，通人性。装备火神炮后战斗力爆表。怨灵的怨结给波奇带很实用。

💜

蕾娜

隐藏NPC

紫发琥珀色眼睛的神秘女人。贯穿二周目全剧情任务链，出现在每一个城镇，又消失在风沙中。

// WORLD MAP

冒险地图

起点

瑞特村

冒险的起点。搜刮家里、对话修车工接任务、去右边工厂拿1号车。酒吧里有一个紫发的女人在等你。

第二站

燧火镇

更新装备、打井下污染源鳄鱼、接楠木博士任务制作奇迹药剂。转职中心在这里，5000金币转职。

第三站

月牙湾镇

获取吉普车钥匙的小镇。附近有金蚁（500G/只），左边森林进入蚂蚁洞穴打亚当蚁。

第四站

克尔镇

收机械师的地方。南边打扎亚斯拿吉普车。附近刷怪升级快、打钱也快。巨炮是第一个硬仗。

中期核心

冶铁镇

停留最久的城镇。下水道拿4号车、蜈蚣洞刷C装置、打马歇尔。全员练到30级的大本营。

恐怖

鬼魂大楼

八层废弃大楼。异形掉落暴走细胞、怨灵掉落智灵——制造生物战车的关键材料。楼顶有万能钥匙。

海洋

锈色少女

搁浅在沙漠中的巨型邮轮。加入欢乐神教后可以使用传送装置进入异世界。可以买到生物引擎。

后期

利特港

海边渔港。买船出海、打U鲨、电波塔拿图纸、打德洛克。红发小女孩的支线催人泪下。

机甲

罗德镇

高达制造工厂所在地。强袭型或重装型二选一。东北工厂打恶霸，东南海域打三个水里BOSS。

终点

亚德拉诺镇

最后的城镇。用勋章换道具、更新最强装备。诺亚基地就在这附近——最终决战的入口。

// HIDDEN QUEST

蕾娜 · 二周目全剧情

💜 一个贯穿全篇的紫发幽灵

蕾娜是本作最具魅力的隐藏NPC。她紫色的头发、琥珀色的眼睛、永远喝不完的那杯酒——她在每一个城镇出现，说一两句莫名其妙的话，然后消失在风沙中。

二周目中，她的全剧情任务链被解锁。你需要追着她的脚步，走遍整个废土世界，才能拼凑出她的完整故事。

"人机情未了——对得起蕾娜吗？人机情未了。"
—— 攻略作者注

她的任务不影响主线，但如果你错过了她——你将错过整个游戏最动人的故事。

// TIPS

攻略要点

01

属性克制优先

武器优先选择属性克制武器收益最大。看怪物抗性百分比——哪个属性百分比高就用哪个属性武器打。

02

C装置是核心

本作有很多C装置增加单穴连射，通过叠加单穴连射来增强输出。后期单穴能打出单回合200+射击数。

03

等变身再杀

诸神黄昏、德洛克等赏金首必须等它们变身/暴走后再击杀，否则不掉落碎片，无法合成终端系统。

04

二周目准备

一周目结束后，传真到瑞特村下车，卸掉除高达之外的战车装备再进入二周目。高达在原来的位置。

05

生物战车材料

鬼楼异形出暴走细胞、怨灵出智灵。这两个材料是制造生物战车的必备品，务必刷到。

06

异世界入口

锈色少女欢乐神教的任务做完后，用能量块开启传送。注意能量块数量有限，进去一次打完再回来。

07

激光炮制作

做完高达才能对话女博士做激光炮。通过镜片不同排列有2个不同形态。4个镜片位置需要仔细寻找。

08

存档！存档！

特殊场景boss连战会有无车可乘的场景。多存档吧骚年——这是废土上最重要的生存法则。
