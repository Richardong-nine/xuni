// LORE

灰烬纪元 · 大破坏之后的世界

💥

大破坏

超级计算机「诺亚」判定人类是地球最大的病毒，发动毁灭性攻击。文明崩塌，城市变成废墟，辐射尘遮蔽天空数十年。

🏚️

废土求生

幸存者蜷缩在残存的城镇里。变异怪物和失控的战争机器遍布荒野。废铁是硬通货，活着就是最大的奢侈。

🎯

猎人

驾驶战车穿越废土，猎杀悬赏怪物换取赏金。猎人既是冒险者也是佣兵——每一辆战车上都写满了故事。

🖥️

诺亚

创造这一切的超级计算机。它在暗处注视着每一个猎人，操控着废土上的一切。它的真正目的，没有人知道。

// GAMEPLAY

核心玩法

🚗

战车改装系统

本作最核心的玩法。获取战车、改装底盘、更换主炮副炮S-E、升级引擎装甲。每辆战车都有独特的底盘特性，通过叠加C装置的单穴连射来打出爆炸伤害。

  • 底盘改装：水鬼车破坏射击、吉普车副炮连射、灰狼车S-E齐射
  • C装置系统：单穴连射是本作特色，后期单回合200+射击数
  • 沙漠之舟、生物战车为后期核心战力
  • 高达机甲：二选一制造，强袭型或重装型
⚔️

人类战斗系统

没有战车的时候，人类也要战斗。战士扔手雷上debuff，摔跤手横岗+架势+台风三连击打输出，护士负责治疗。阵容搭配决定生死。

  • 战士+摔跤手+护士：经典人战阵容
  • 猎人+艺术家+摔跤手：辅助吹军号金粉，摔跤手3回合爆发
  • 活用道具：军号增加全体属性、水母烟雾弹致盲、强力造云素降防
  • 怨灵掉落的「怨结」：圣临技能解除全体debuff+回复20%血
🧬

生物战车

在生物研究所用暴走细胞和智灵制造。霸王龙、三角龙、猛犸象三选一。生物战车能自我修复，是后期的核心战力。需要从鬼魂大楼的异形和怨灵身上获取材料。

  • 霸王龙车：攻击力最强
  • 三角龙车：防御力最好
  • 猛犸象车：践踏攻击，嘎嘎爽

车战推荐阵容

HUMAN COMBAT

人战推荐阵容

战士摔跤手护士

经典人战阵容。战士丢手雷上debuff百分比掉血，摔跤手横岗+架势+台风（冷气3连击）/肘击（火焰4连击）三回合爆发一次，护士负责治疗。武器选攻击力高的，多连/范围武器不触发连击。

// WORLD MAP

冒险地图

起点

瑞特村

冒险的起点。搜刮家里、对话修车工接任务、去右边工厂拿1号车。酒吧里有一个紫发的女人在等你。

第二站

燧火镇

更新装备、打井下污染源鳄鱼、接楠木博士任务制作奇迹药剂。转职中心在这里，5000金币转职。

第三站

月牙湾镇

获取吉普车钥匙的小镇。附近有金蚁（500G/只），左边森林进入蚂蚁洞穴打亚当蚁。

第四站

克尔镇

收机械师的地方。南边打扎亚斯拿吉普车。附近刷怪升级快、打钱也快。巨炮是第一个硬仗。

中期核心

冶铁镇

停留最久的城镇。下水道拿4号车、蜈蚣洞刷C装置、打马歇尔。全员练到30级的大本营。

恐怖

鬼魂大楼

八层废弃大楼。异形掉落暴走细胞、怨灵掉落智灵——制造生物战车的关键材料。楼顶有万能钥匙。

海洋

锈色少女

搁浅在沙漠中的巨型邮轮。加入欢乐神教后可以使用传送装置进入异世界。可以买到生物引擎。

后期

利特港

海边渔港。买船出海、打U鲨、电波塔拿图纸、打德洛克。红发小女孩的支线催人泪下。

机甲

罗德镇

高达制造工厂所在地。强袭型或重装型二选一。东北工厂打恶霸，东南海域打三个水里BOSS。

终点

亚德拉诺镇

最后的城镇。用勋章换道具、更新最强装备。诺亚基地就在这附近——最终决战的入口。

// WALKTHROUGH

攻略流程

01 // 瑞特村

冒险起点

搜刮家里 → 酒吧对话 → 修车工接任务 → 右边工厂拿1号车

搜刮家里（1楼右边房间需万能钥匙，箱子是宝石镜）。去酒吧对话每个人（二周目旅馆对话蕾娜）。对话修车工接刷10个废铁任务，做完去右边工厂拿1号车。买10个火箭礼花备用。

02 // 燧火镇

更新装备

更新装备 → 打鳄鱼 → 接楠木博士任务

出村向左再向上，途经商入秘营地。到燧火镇更新装备买传真。右上调查井下去打污染源鳄鱼（收丹妮娜再打）。左边大楼需完成楠木博士任务才能进。大楼右上方营地有波奇收。

03 // 月牙湾镇

吉普车钥匙

收丹妮娜 → 拿吉普车钥匙 → 蚂蚁洞穴

出镇向上过关卡遇丹妮娜，组队打败机器人后单挑，她加入。到月牙湾右边过桥向下进小屋拿吉普车钥匙。附近有金蚁（500G/只）。左边森林进蚂蚁洞穴打亚当蚁。

04 // 克尔镇

收机械师 · 拿吉普车

收机械师 → 扎亚斯拿车 → 巨炮硬仗

进镇收机械师。南边下楼打扎亚斯（3W血）拿吉普车，战士10级学投掷，机械师坐车用烟尘炮弹致盲。至此战队成型。附近刷怪升级快、打钱也快。巨炮是第一个硬仗。

极限过巨炮：主角15级出猎人之眼+装甲化，机械师烟尘炮弹致盲+装甲恢复，先集火打掉一个。
05 // 冶铁镇

中期大本营

下水道拿4号车 → 蜈蚣刷C装置 → 打马歇尔 → 全员30级

停留最久的城镇。进镇先去下水道拿4号车（二周目蕾娜在下水道路口）。蜈蚣掉落C装置（一穴连射+1）必刷。马歇尔有难度，转职护士和摔跤手好打。打完调查电脑开栏杆。

06 // 鬼魂大楼

刷生物战车材料

异形出暴走细胞 → 怨灵出智灵 → 楼顶万能钥匙

8层废弃大楼。先去右下地下一层接任务。推油桶开正门。每层仔细开箱子柜子。8层中间做探鬼帽。先上楼顶打异形拿万能钥匙，再戴探鬼帽打怨灵。怨灵怕火和冰，有成佛道具很好打。

鬼楼所有BOSS掉落物都要刷出来！暴走细胞+智灵是制造生物战车的必备材料。
07 // 锈色少女

异世界传送

加入欢乐神教 → 打教主 → 能量块开传送

搁浅的巨型邮轮。加入欢乐神教，完成任务后打教主（人战得能量块），坐电梯打绝对防御神话（车战）。用能量块开传送进异世界。能量块数量有限，进去一次打完再回来。

08 // 利特港

买船出海

打U鲨 → 买船 → 电波塔拿图纸 → 打德洛克

红发小女孩支线催人泪下（进港1小时内买花）。打U鲨后买船出海。电波塔4层有MK和PB图纸。打德洛克必须等暴走后再杀，否则不掉落碎片。附近有大鸟诸神黄昏。

德洛克暴走后光束和冷气效果好。必须等暴走再杀，否则无法合成终端系统！
09 // 罗德镇

高达机甲

东南工厂造高达 → 东北工厂打恶霸 → 海域打3个BOSS

高达二选一制造。材料：高科技废铁（离子坦克）、金属外壳（沙漠3个战车BOSS）、能源核心（鬼楼负1楼守卫智能人）、杀戮系统（冥狱哨卡持刀守卫）。做完高达才能做激光炮。

10 // 最终决战

诺亚基地

收集24碎片 → 合成终端 → 4形态连战

用ID卡开大门。终端需全部赏金首碎片合成。BOSS每回合随机变化抗性，上BUFF磨血（火冰每回合掉1-2W），吃致盲。4连战中间不停，建议100级+。打完得退出按钮和未来芯片。

4形态连战中间不停！多存档。机械师注意修车，艺术家加闪避清小怪致盲。
// HIDDEN QUEST

蕾娜 · 二周目全剧情

💜 一个贯穿全篇的紫发幽灵

蕾娜是本作最具魅力的隐藏NPC。她紫色的头发、琥珀色的眼睛、永远喝不完的那杯酒——她在每一个城镇出现，说一两句莫名其妙的话，然后消失在风沙中。

二周目中，她的全剧情任务链被解锁。你需要追着她的脚步，走遍整个废土世界，才能拼凑出她的完整故事。

"人机情未了——对得起蕾娜吗？人机情未了。"
—— 攻略作者注

她的任务不影响主线，但如果你错过了她——你将错过整个游戏最动人的故事。

// TIPS

攻略要点

01

属性克制优先

武器优先选择属性克制武器收益最大。看怪物抗性百分比——哪个属性百分比高就用哪个属性武器打。

02

C装置是核心

本作有很多C装置增加单穴连射，通过叠加单穴连射来增强输出。后期单穴能打出单回合200+射击数。

03

等变身再杀

诸神黄昏、德洛克等赏金首必须等它们变身/暴走后再击杀，否则不掉落碎片，无法合成终端系统。

04

二周目准备

一周目结束后，传真到瑞特村下车，卸掉除高达之外的战车装备再进入二周目。高达在原来的位置。

05

生物战车材料

鬼楼异形出暴走细胞、怨灵出智灵。这两个材料是制造生物战车的必备品，务必刷到。

06

异世界入口

锈色少女欢乐神教的任务做完后，用能量块开启传送。注意能量块数量有限，进去一次打完再回来。

07

激光炮制作

做完高达才能对话女博士做激光炮。通过镜片不同排列有2个不同形态。4个镜片位置需要仔细寻找。

08

存档！存档！

特殊场景boss连战会有无车可乘的场景。多存档吧骚年——这是废土上最重要的生存法则。

// BOSS STRATEGY

BOSS攻略

诸神黄昏（大鸟）

碎片 ω · 终端系统必备

沙漠随机遇敌。必须等它变身觉醒后再杀，否则不掉落碎片，无法合成终端系统。沙漠附近有几率遇到。

等变身再杀

德洛克

碎片 ψ · 电波塔右上方山洞

打掉一半血时暴走，提升攻击和抗性。暴走后光束和冷气效果好。必须等暴走后再杀，否则无掉落物。

等暴走再杀 · 光束/冷气

怨灵

碎片 σ · 鬼魂大楼8层

怕火和冰。摔跤手"火焰歌舞伎"好用。有成佛道具时，打掉2-3W血后自动使用，一次性掉30W血。掉落怨结（圣临技能）。

怕火/冰 · 成佛秒杀

蜈蚣

碎片 ι · 冶铁镇北边山洞

一层直接走到二层掉到最下层。主角巨像重炮+狙击+电光火石高输出。掉落C装置（一穴连射+1）必刷。

高输出速杀 · C装置必刷

恶霸

罗德镇东北工厂

先打6个恶霸小弟（无法逃跑，小弟不断提升攻击力）。恶霸会"内在潜能"每回合自己掉血12%。上火焰或冷气BUFF掉血快。

火焰/冷气BUFF · 致盲

诺亚（最终BOSS）

4形态连战 · 诺亚基地

每回合随机变化抗性。上BUFF耗血（火冰每回合1-2W可叠加）。4连战中间不停。半血会召唤小兵。建议100级+，装备BUFF副炮主炮。

吃致盲 · 上BUFF磨血

二周目攻略

进入二周目准备

一周目结束后，使用退出按钮可选择进入二周目。进入前务必做好以下准备：

  • 传真到瑞特村内下车
  • 卸掉除高达之外的战车装备（高达在原来的位置）
  • 人类和狗子装备可不脱，很快就能召集到队友
  • 装备为原本穿戴装备

二周目新增内容

二周目添加蕾娜全剧情任务链（详见蕾娜章节）。新增多个赏金首和强化材料来源：
